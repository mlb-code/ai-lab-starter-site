01

מי נגד מי — מורה נבוכים לעולם ה-AI

מפת השחקנים: Claude, Gemini, Grok, ChatGPT. איזה כלי מתאים לאיזו משימה.

+
במה נעסוק
  • ארבעת השחקנים העיקריים בעולם ה-LLMs וההבדלים ביניהם
  • למה Claude הוא הכלי המרכזי בקורס
  • איזה כלי מתאים לאיזו משימה — כלל אצבע פשוט
  • מפת הכלים שילוו אותך לאורך הקורס
02

הכלים שיבנו בשבילך

Cursor, Claude Desktop, ו-Terminal — שלוש סביבות עבודה שמרכיבות את ה-stack שלך.

+
במה נעסוק
  • מה זה Terminal ולמה כל מתכנת מכיר את המונח
  • Cursor — העורך החכם שתעבוד איתו ברוב הקורס
  • Claude Desktop ושלושת המצבים שלו: Chat · Cowork · Code
  • התקנה חכמה: Cursor מתקין את הכל בשבילך, בלי טרמינל
03

GitHub — בית הקוד שלך

איפה הקוד שלך גר, איך לדחוף אותו לענן, ואיך לפרסם אתר חינמי תוך 5 דקות.

+
במה נעסוק
  • Git לעומת GitHub — מה ההבדל ולמה אתה צריך את שניהם
  • שלושת הפעולות שתעשה כל יום: commit, push, pull
  • Repository, README, ומה הם אומרים על הפרויקט שלך
  • GitHub Pages — איך להעלות אתר חינמי לאוויר
04

בונים את ה'פנים' — דף נחיתה ראשון

מהרעיון לאתר חי באוויר. אנטומיה של דף נחיתה מקצועי, פרומפט מנצח, ופרסום.

+
במה נעסוק
  • אנטומיה של דף נחיתה מקצועי — Hero, About, CTA, Footer
  • הפרומפט המנצח: איך לבקש מ-Claude לבנות דף נחיתה לעסק שלך
  • החלפת תוכן, צבעים, ותמונות — בלי לעבוד עם קוד
  • פרסום ב-GitHub Pages — אתר חי באוויר עם URL לשיתוף
05

ה'מוח' של האתר — חיבור ל-Google Sheets

הופכים אתר מ"דף יפה" למערכת חיה — עם טופס לידים מחובר לגיליון.

+
במה נעסוק
  • למה Google Sheets זה בסיס הנתונים הכי פשוט בעולם
  • Apps Script — השרת החינמי של גוגל שמחבר את הכל
  • חיבור טופס באתר ל-Sheet — לידים נכנסים אוטומטית
  • אוטומציה: התראה במייל בכל ליד חדש
06

אוטומציות אישיות — הצוות הסודי שלך

חמש אוטומציות שכל אחד יכול לבנות. עוזר בוקר, מעקב הוצאות, ועוד.

+
במה נעסוק
  • האנטומיה של כל אוטומציה: Trigger → Steps → Output
  • 5 דוגמאות פרקטיות: עוזר בוקר, מעקב הוצאות, ימי הולדת, משלוחים, סיכום קבוצות
  • איך להבין מה כדאי לאוטמט בחיים שלך
  • המשימה: לבנות בעצמך עוזר בוקר אישי

שאלות נפוצות

כל מה שחששתם לשאול

אם השאלה שלכם לא כאן — שלחו הודעה בוואטסאפ ותקבלו תשובה אישית באותו היום.

כן, וזה בדיוק קהל היעד. הקורס נבנה לאנשים בלי רקע טכני — נסביר כל מושג חדש, נראה כל פעולה צעד-צעד. אנשים בני 60+ עוברים את הקורס ומסיימים עם כלים אמיתיים.
השיעור עצמו 45-55 דקות, פעם בשבוע. בנוסף שיעורי בית של כשעה (אופציונליים — מי שעוקב לאט יותר יכול לעשות אותם בקצב שלו). סה"כ ~2-3 שעות בשבוע.
כל שיעור מוקלט ועולה לאזור האישי תוך 24 שעות. בנוסף, המצגת המלאה זמינה לעיון בכל עת. אפשר לחזור על שיעור גם חודש אחרי שעבר.
כן. יש קבוצת WhatsApp עם תמיכה ישירה ממאיר. אפשר לשאול כל שאלה. בנוסף, תוך כדי השיעורים בזום יש זמן ייעודי לשאלות.
רק שניים: Claude Pro (חיוני) ב-$20 לחודש, ו-Cursor (מספיק בגרסה החינמית). שאר הכלים — GitHub, Google Sheets, Apps Script — חינמיים לחלוטין.
Mac (M1 ומעלה) או Windows 10+ עם 16GB RAM ומעלה, מעבד i5 דור 10+ או שווה ערך. רוב המחשבים האישיים מ-5 השנים האחרונות עומדים בדרישות.
כן. הקהילה ממשיכה לאחר הקורס — סטודנטים שעברו את Starter ממשיכים לבנות יחד, לשתף, ולהיעזר. בנוסף, יש קורסים מתקדמים יותר לאלו שרוצים להעמיק.
השאירו פרטים — נחזור אליכם, נתאם שיחה קצרה (15 דקות), ונבדוק יחד אם הקורס מתאים. אם לא — נכוון אתכם לאן שכן מתאים.
